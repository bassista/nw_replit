# Cibi · Aggiungi/Modifica

## Campi
- Nome, Categoria
- Calorie, Proteine, Carboidrati, Grassi, Fibre
- Grammi per porzione

## Validazioni
- Numeri positivi
- Categoria esistente

## Flusso
- Apri dialog → compila campi → salva
- Se esiste, aggiorna voce; altrimenti crea nuova
