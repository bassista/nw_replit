# Homepage

La homepage fornisce una panoramica rapida dello stato giornaliero e accessi alle principali funzionalità.

## Azioni principali

## Dipendenze UI
- React 18 + Tailwind CSS
- Componenti shadcn/ui (cards, tabs, toast)

## Design
- **CTA primarie** in verde brand `#22c55e` (HSL 142/71/45)
- **Card** per riepiloghi giornalieri: `card-elevated` per evidenza, `card-filled` per sezioni secondarie
- **Spaziatura** coerente con ritmo verticale; separatori leggeri per blocchi
- **Icone** lucide 16–24px per azioni rapide
- **Status bar** chiara con sfondo brand su Android; splash minimale
- **Motion** discreta (transizioni brevi, feedback toast)
